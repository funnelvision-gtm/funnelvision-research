# FunnelVision — LinkedIn Prospect Data

## Purpose
Detailed prospect data from LinkedIn Sales Navigator for competitor reviewer profiles. Tracking both review-time and current roles/companies to identify job changes and new opportunities.

## Format
Each entry tracks:
- **Source**: Which competitor review file and review number
- **Review title / Review company**: What they were when they wrote the review
- **Current title / Current company**: Where they are now (from LinkedIn)
- **Location**: Current location
- **LinkedIn URL**: Direct profile link
- **Notes**: Job changes, company details, anything notable
